🎮 Arduino Mini Arcade - Retro Atari Games
🕹️ How I Brought Classic Atari Games Back to Life Using Arduino
📌 Project Overview

ده مشروع Mini Arcade Console عملته باستخدام Arduino، رجعت فيه روح ألعاب الأتاري القديمة لكن بتصميم حديث وصغير 👀
🎮 الألعاب الموجودة

المشروع فيه مجموعة ألعاب كلاسيك مستوحاة من عصر الأتاري:

🚗 Car Avoider

اهرب من العربيات اللي جاية عليك
كل ما تفضل عايش أكتر → السكور يزيد

🧱 Breakout

اكسر الطوب بالكرة
لو الكرة وقعت = خسرت 😬

🧩 Tetris

رتّب الأشكال عشان تملى الصفوف
لعبة الإدمان الأبدي 😄

🐍 Snake

كل الأكل وكبّر الثعبان
بس خليك حذر من نفسك 👀

👾 Space Invaders

اضرب الفضائيين قبل ما يوصلوا لك
Classic بكل معنى الكلمة 🔥

🐤 Flappy Bird

اضغط عشان تطير
لو خبطت = انتهيت 💀

🏓 Ping Pong

تحكم في المضرب وارجع الكرة
Simple بس صعب تتقنها 😏

🧰 المكونات (Hardware)
Arduino UNO / Nano
LED Matrix 8x8 (MAX7219)
Joystick Module
Buzzer
Push Button 
Potentiometer
